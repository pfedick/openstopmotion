Please note:

  - the File "*-i386.deb" is for 32-Bit Ubuntu
  - the File "*-amd64.deb" is for 64-Bit Ubuntu
  
To install the package, simply download the file. In most cases, the package
manager will open automatically and you can press the "Install"-Button.

If it does not oben, double click on the file from the filemanager, or
install it manually with dpkg, eg:

   sudo dpkg -i OpenStopMotion-0.6.3-Ubuntu-12.04-amd64.deb
