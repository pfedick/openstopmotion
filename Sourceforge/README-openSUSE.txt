Please note:

  - the File "*.i686.rpm" is for 32-Bit openSUSE systems
  - the File "*.x86_64.rpm" is for 64-Bit openSUSE systems
  
To install the package, simply download the file, and install
it with zypper, eg:

   sudo zypper install OpenStopMotion-0.6.4-1.suse.i686.rpm
   
The Package was compiled on OpenSuse 12.3. If it does not run, download
the source rpm from the parent directory and try to build your own
binary package.
