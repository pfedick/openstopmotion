Please note:

  - the File "*fc18.i686.rpm" is for 32-Bit Fedora 18 systems
  - the File "*fc18.x86_64.rpm" is for 64-Bit Fedora 18 systems
  
To install the package, simply download the file, and install
it with yum, eg:

   su root
   yum install OpenStopMotion-0.6.4-1.fc18.x86_64.rpm
   
The Package was compiled on Fedora 18. If it does not run, download
the source rpm from the parent directory and try to build your own
binary package.
