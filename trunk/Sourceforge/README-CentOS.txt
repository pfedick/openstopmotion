Please note:

  - the File "*.i386.rpm" is for 32-Bit systems
  - the File "*.x86_64.rpm" is for 64-Bit systems
  
To install the package, simply download the file, and install
it with yum, eg:

   su root
   yum install OpenStopMotion-0.6.3-1.el6.x86_64.rpm
   
The Package was compiled on CentOS 6, so it should work on all compatible
systems (eg. RedHat and Fedora). If it does not run, download the source
rpm from the parent directory and try to build your own binary package.
